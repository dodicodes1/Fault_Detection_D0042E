## READ ME — Fault Detection in District Heating
This project was developed for the course AI and Machine Learning for Engineering Studies (D0042E).
The goal of the lab was to create a machine-learning-based fault detection system for a simulated district heating system using live MQTT sensor data.

The system monitors a three-room house connected to a district heating network and detects faults in real time using a trained neural network model.

The project includes:

* Data exploration and visualization
* Feature engineering
* Model training and evaluation
* Real-time fault detection using MQTT streaming data

## How to use

Run the jupyter file "Task1-3_2_labels.ipynb" and then the python file "Task4_neural_network.py" or "Task4_random_forest.py" depending on which model you want to connect to the live brooker.
